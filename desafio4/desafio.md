# Lista de Atividades - Uso de Listas em Programação

Saber como usar listas é um dos conceitos fundamentais da programação e do desenvolvimento de software.

Pensando nisso, criamos uma lista de atividades (não obrigatórias) focada em prática para melhorar ainda mais sua experiência de aprendizagem.

## Bora praticar então?

### Desafios
1. **Lista Genérica:**
   - Crie uma lista vazia, com o nome `listaGenerica`.

2. **Lista de Linguagens de Programação:**
   - Crie uma lista de linguagens de programação chamada `linguagensDeProgramacao` com os seguintes elementos: `'JavaScript', 'C', 'C++', 'Kotlin' e 'Python'`.

3. **Adicionar Elementos à Lista:**
   - Adicione à lista `linguagensDeProgramacao` os seguintes elementos: `'Java', 'Ruby' e 'GoLang'`.

4. **Primeiro Elemento da Lista:**
   - Crie uma lista com 3 nomes e exiba no console apenas o primeiro elemento.

5. **Segundo Elemento da Lista:**
   - Crie uma lista com 3 nomes e exiba no console apenas o segundo elemento.

6. **Último Elemento da Lista:**
   - Crie uma lista com 3 nomes e exiba no console apenas o último elemento.

Caso precise de ajuda, opções de solução das atividades estarão disponíveis na seção “Opinião da pessoa instrutora”.
